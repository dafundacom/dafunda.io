### Absolute Import is a great way to clean up your imports

## Example

This is the usual way of importing with `..` operator to go back 1 folder:

```jsx
import Button from "../../components/Button"
```

And this is the clean import using absolute import:

```jsx
import Button from "@/components/Button"
```

---

# For Next.js

## Add this to root with the filename of `jsconfig.json`

```jsx
{
  "compilerOptions": {
    "jsx": "preserve",
    "baseUrl": ".",
    "paths": {
      "@/*": ["*"]
    }
  },
  "exclude": ["node_modules", ".next"]
}
```

If you store your files inside src folder change baseUrl to ./src.

```jsx
{
  "compilerOptions": {
    "jsx": "preserve",
    "baseUrl": "./src",
    "paths": {
      "@/*": ["*"]
    }
  },
  "exclude": ["node_modules", ".next"]
}
```

# For Next.js TypeScript

## Add to `tsconfig.json`

```js
{
  "compilerOptions": {
    "baseUrl": "./src",
    "paths": {
      "@/*": ["*"]
    }
  },
}
```

# For Create React App

## Add this to root with the filename of `jsconfig.json`

```jsx
{
  "compilerOptions": {
    "baseUrl": "./src",
    "jsx": "preserve",
    "paths": {
      "@/*": ["./src/*"],
      "@/components/*": ["./components/*"],
      "@/layouts/*": ["./layouts/*"],
      "@/routes/*": ["./routes/*"]
    }
  },
  "exclude": ["node_modules", "build"]
}
```

## And in craco.config.js

```js
const path = require("path")

module.exports = {
  // ...existing code
  webpack: {
    alias: {
      "@": path.resolve(__dirname, "src"),
    },
  },
}
```
